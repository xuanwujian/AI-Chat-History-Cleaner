# AI Chat History Cleaner - AI 聊天记录清理器

一款用于一键删除各大 AI 对话平台历史记录的 Chrome 扩展程序。

## ✨ 功能特性

- **多平台支持**：支持 ChatGPT、Claude、Gemini、DeepSeek、豆包、通义千问、Kimi、智谱清言、腾讯元宝、Perplexity、Microsoft Copilot、Poe 等 14 个主流 AI 平台
- **双模式删除**：支持 API 批量删除和 DOM 逐条删除两种模式
- **智能检测**：自动识别各平台的 UI 变化，适配最新界面
- **完全本地**：所有操作在本地浏览器执行，无数据上传，保护隐私

## 🔧 安装方法

1. 打开 Chrome 扩展管理页面：`chrome://extensions/`
2. 启用右上角的 **"开发者模式"**
3. 点击 **"加载已解压的扩展程序"**
4. 选择扩展文件夹（包含 `manifest.json` 的目录）

## 📖 使用方法

1. **打开 AI 网站**：在 Chrome 中访问任意支持的 AI 对话平台
2. **启动清理**：点击浏览器工具栏中的扩展图标
3. **开始删除**：点击 "开始删除历史记录" 按钮
4. **监控进度**：实时查看删除进度和日志

### 高级选项

- **停止清理**：点击 "停止删除" 按钮可随时中断
- **重新检测**：点击 "重新检测" 刷新页面状态

## ⚠️ 注意事项

1. **删除不可恢复**：清理操作是永久性的，请在操作前确认
2. **逐条删除需要时间**：大量历史记录可能需要较长时间
3. **部分受保护对话**：某些被锁定或受保护的特殊对话可能无法删除
4. **网络稳定性**：建议在网络稳定的环境下使用
